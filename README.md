# :earth_americas: Interview practice app

A simple Streamlit app that takes user's provided URL (job ad) and connects to Gemini flash 2.5 model to get the job add summary, that includes technical skills, practice question per each technical skill and link to source material per each technical skill to learn that skill.

[![Open in Streamlit](https://vigilant-meme-46jxrjwgq5xhjqpw-8502.app.github.dev/)

### How to run it on your own machine

1. Install the requirements (see list oin the requiremnets.txt file)

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
